# Instagram Post Images

Place your Instagram post images in this folder.

## Instructions

1. Download images from your Instagram posts
2. Save them in this folder with descriptive names (e.g., `post1.jpg`, `project-launch.png`)
3. Update `content/instagram/posts.md` to reference these images using `/instagram-posts/your-image-name.jpg`

## Image Requirements

- **Format**: JPG, PNG, or WebP
- **Recommended size**: 800x800 pixels or larger (square images work best)
- **File size**: Keep under 500KB for fast loading
- **Aspect ratio**: 1:1 (square) recommended

## Example

If you save an image as `my-post.jpg` in this folder, reference it in `posts.md` as:
```
Image: /instagram-posts/my-post.jpg
```
