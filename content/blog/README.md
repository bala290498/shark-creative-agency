# Blog Content

This folder contains markdown files for blog posts. Each markdown file should follow this structure:

## Frontmatter (YAML)

```yaml
---
title: "Your Blog Post Title"
excerpt: "A brief description of the post"
author: "Author Name"
date: "YYYY-MM-DD"
readTime: "X min read"
category: "Category Name"
image: "Image URL"
---
```

## Content

The main content should be written in Markdown format below the frontmatter.

## Categories

Available categories:
- Web Design
- Branding
- Marketing
- SEO
- Content

## File Naming

Use kebab-case for filenames, e.g., `my-blog-post-title.md`
