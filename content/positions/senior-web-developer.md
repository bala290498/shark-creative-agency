## About the Role

We are looking for a Senior Web Developer who is passionate about building modern, scalable web applications. You'll work on diverse projects, from custom web applications to e-commerce platforms, and collaborate closely with our design and marketing teams.

## Key Responsibilities

- Develop and maintain high-quality web applications using React, Next.js, and TypeScript
- Collaborate with designers to implement pixel-perfect UI/UX designs
- Write clean, maintainable, and well-documented code
- Optimize applications for maximum speed and scalability
- Participate in code reviews and mentor junior developers
- Work with the team to define technical requirements and architecture

- Opportunities for professional growth and development
- Work on exciting projects with cutting-edge technologies
